# PX4 Drone Delivery VTOL

## Setup

```
git clone git@github.com:mmousaei/PX4-Autopilot.git --recursive
cd PX4-Autopilot
git checkout mmousaei/cmu_add_v2
git submodule sync
git submodule update --init --recursive --remote
make px4_sitl_ctrlalloc gazebo_needle
```
